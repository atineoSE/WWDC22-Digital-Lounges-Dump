Michael Patrick J (Apple)

*Thread Reply:* We provide a Geometry field on the model reconstruction Request that will allow you to specify the cropping region during the model creation step. Also note that this year we added the ability to specify an OrientedBoundedBox in the request to allow a more general cropping volume.
